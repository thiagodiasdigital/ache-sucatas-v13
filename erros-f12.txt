Mensagem: TypeError: Map is not a constructor
Este é o erro principal que está impedindo a renderização da página. O problema ocorre dentro de um hook useMemo no componente MapContent. A causa provável é:

O código está tentando usar new Map() (estrutura de dados JavaScript), mas algo está sobrescrevendo ou conflitando com o construtor nativo Map
Pode haver um import incorreto onde uma biblioteca de mapas (como Leaflet ou Google Maps) está sendo importada com o nome Map, conflitando com o Map nativo do JavaScript
Ou o contrário: o código espera usar um componente de mapa de uma biblioteca, mas está recebendo algo diferente

Sugestão de correção: Verifique os imports no arquivo AuctionMap.tsx e certifique-se de que não há conflito de nomes entre o Map nativo do JavaScript e componentes/classes de bibliotecas de mapas.
